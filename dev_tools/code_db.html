üì¶ Required Base Template

Base HTML Structure

This is the mandatory HTML structure that must exist in every page for the design system to work correctly. Includes all required CSS, JavaScript, fonts, and meta tags.

Required Files & Dependencies
  • CSS: ../css_src/index.css (Main styles)
  • CSS: lessons.css (Lesson-specific styles)
  • Icons: Boxicons (via CDN)
  • JavaScript: ../script_src/index.js (Main scripts)
  • Viewport: Responsive meta tag required

üìê Layout Sections

Main Content Section

The primary content wrapper that provides proper spacing and layout for page content. This is where all lesson content and components should be placed.

Must Be Used Inside

Must be placed within <main class="main"> after the header.

Section Template 
<section class="section" style="padding-block: 4rem;">
    <div class="container">
        <!-- Your content here -->
        <h1 class="title-large">Your Title</h1>
        <p class="text-medium font-normal">Your content...</p>
    </div>
</section>
Warning

‚ö†Ô∏è Container class is required: All content must be wrapped in .container for proper padding and max-width.

‚ö†Ô∏è Padding styles: Use style="padding-block: Xrem" to control vertical spacing between sections.

Complete Practice Section with Wrapper

Interactive form for quizzes or practice exercises within lessons. Requires the practice wrapper for proper padding and styling.

Required Parent Structure

The form MUST be wrapped in .lesson-practice for proper padding (2rem), background, and border radius.

Complete Practice Section Template 
<div class="lesson-practice">
    <h3 class="lesson-practice-title title-small">Try It Yourself</h3>
    <p class="text-base font-normal">
        Instructions for the practice exercise.
    </p>
    
    <form class="lesson-form">
        <div class="lesson-form-group">
            <label for="question1" class="lesson-form-label">
                1. Question text here?
            </label>
            <input type="text" id="question1" name="question1" class="lesson-form-input"
                placeholder="Your answer" />
        </div>
        
        <div class="lesson-form-group">
            <label for="code-practice" class="lesson-form-label">
                2. Code practice question
            </label>
            <textarea id="code-practice" name="code-practice" class="lesson-form-textarea"
                rows="4" placeholder="Your code here"></textarea>
            <span class="lesson-form-hint">Additional hint text</span>
        </div>
        
        <div class="lesson-form-group">
            <label class="lesson-form-label">
                3. Multiple choice question
            </label>
            <div class="lesson-form-radio-group">
                <label class="lesson-form-radio">
                    <input type="radio" name="question3" value="option1" />
                    <span>Option 1</span>
                </label>
                <label class="lesson-form-radio">
                    <input type="radio" name="question3" value="option2" />
                    <span>Option 2</span>
                </label>
            </div>
        </div>

            <button type="submit" class="btn btn-primary">Submit Answers</button>
            <button type="reset" class="btn btn-darken">Reset</button>

        <div class="lesson-form-message lesson-form-success" style="display: none;">
            <i class='bx bx-check-circle'></i>
            <span>Success message</span>
        </div>
        
        <!-- Error Message (hidden by default) -->
        <div class="lesson-form-message lesson-form-error" style="display: none;">
            <i class='bx bx-error-circle'></i>
            <span>Error message</span>
        </div>
    </form>
</div>
Critical Form Requirements

‚ö†Ô∏è .lesson-practice wrapper required: Without this wrapper, the form lacks proper padding and background styling.

‚ö†Ô∏è Form group spacing: Each .lesson-form-group provides 1.5rem bottom margin automatically.

‚ö†Ô∏è JavaScript required: Form validation and message display requires the JavaScript from the base template.
